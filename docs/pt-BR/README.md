# Documentação em Português (Brasil)

Esta pasta contém as traduções para Português do Brasil dos documentos
canônicos do projeto deEHR.

A documentação canônica é mantida em **inglês**
([../architecture/](../architecture/),
[../security/](../security/), [`../../README.md`](../../README.md), etc.).
Esta convenção segue a política de documentação do repositório (en
canônico + sufixo `.pt-BR` sempre cross-linked).

> **Status:** Em construção (Fase 0). A tradução dos documentos da Fase 0
> está sendo rastreada pela [issue #9](https://github.com/brunocampos-ssa/deEHR/issues/9).
