# rnds-connector

Isolated module for Brazil's **RNDS** (Rede Nacional de Dados em Saúde)
integration. Owns the ICP-Brasil certificate handling and the RNDS-defined
FHIR profile mapping so RNDS specifics stay out of the core platform.

See the *RNDS & Government Integration* section in the
[README](../../README.md) and the FHIR profile selection
([#6](https://github.com/brunocampos-ssa/deEHR/issues/6)).

> **Status:** Phase 4 — not yet implemented.
